## Funcionalidades

- **Página Inicial**: Apresentação da organização, missão, visão, valores e informações de contato
- **Página de Projetos**: Divulgação de oportunidades de voluntariado e formas de doação
- **Página de Cadastro**: Formulário completo para cadastro de voluntários e doadores

## Tecnologias Utilizadas

- HTML5 Semântico
- CSS3 (Grid, Flexbox, Variáveis CSS)
- JavaScript (Máscaras de input, Validação de formulários)
- Design Responsivo (Mobile-first)

## Como Executar

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` em um navegador web
3. Navegue entre as páginas usando o menu de navegação

## Requisitos Atendidos

- [x] Estrutura HTML5 semântica com 3 páginas
- [x] Hierarquia de títulos lógica e consistente
- [x] Formulário complexo com validação nativa
- [x] Máscaras de input para CPF, telefone e CEP
- [x] Design responsivo (mobile-first)
- [x] Estrutura de pastas organizada
